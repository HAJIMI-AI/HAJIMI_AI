✦ 核心亮点

子应用平台 · 快速赋予 AI 能力

哈基米的子应用体系，让任何 Web 应用几分钟内接入 AI—— 只需一个配置文件，应用即刻获得完整的 AI Agent 调用、模型管理、文件操作等能力。
桌面端承载引擎，MQTT 打通远程，子应用只管业务逻辑。

完整的 AI 工作台

从模型管理到远程通信，覆盖 AI 应用开发的全链条。

多模型管理

添加任意兼容 Claude API 的模型端点，设置默认模型，每个账户数据完全隔离。一次配置，随处使用。

技能系统

上传 Claude Agent Skills 作为本地目录，批量导入，元数据展示，作用域过滤。技能可自由组合。

插件包管理

技能 + MCP 配置组合成命名插件包，一键注入 Agent。模块化管理，灵活组合，复用性强。

MQTT Broker

内置 Aedes MQTT Broker，TCP/WebSocket 双协议，可选认证。外部应用远程调度全部平台服务。

多账户隔离

本地账户系统，按 userKey 完全隔离所有数据（模型、会话、技能、KV）。切换账户自动重连 MQTT。

日志 & 文件服务

每日轮转日志系统，可查询/导出。完整文件系统操作，读写、预览、选择、finder 定位。
